# 🏏 IPL Hub

IPL Hub is a modern and responsive cricket web application where users can explore IPL teams, players, match schedules, points tables, and live cricket-related updates in one place.  
Built with a clean UI and smooth user experience for cricket fans. 🚀

---

## ✨ Features

- 🏆 IPL Teams Information
- 👨‍💻 Players Details
- 📅 Match Schedule
- 📊 Points Table
- 🔍 Search Functionality
- 📱 Fully Responsive Design
- ⚡ Fast & User-Friendly Interface

---
